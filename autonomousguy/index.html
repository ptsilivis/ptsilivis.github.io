Why an AI assistant needs automotive context

Embedded automotive work carries heavy domain context — AUTOSAR layering, MISRA C rules, ISO 26262 ASIL logic — that general AI tools do not know. Without it you re-explain the same background on every prompt.

AutonomousGuy encodes that context once, as Agent Skills. The result: a MISRA C AI assistant, an ISO 26262 AI workflow and an AUTOSAR Classic / Adaptive AI agent inside the coding tool you already use — giving consistent, expert-level answers without the re-explaining.

10 skills across the ECU software lifecycle

Mode-aware · Classic + Adaptive

Each skill routes from your input to the right sub-workflow, so one skill handles several related tasks. Start with codebase-analysis: it writes a codebase map the other skills reference. Skills default to Classic AUTOSAR and switch to Adaptive when your input names AP concepts (ara::com, ara::exec, POSIX, service-oriented).

codebase-analysis

Scans the repo and maps SWCs, BSW usage, port interfaces, RTE API calls, signal flow and ASIL zones into a codebase map.

autosar-swc

Component design, port interface definition, SWC code, plain-text diagrams and integration review.

autosar-bsw

BSW config (Com, NvM, Dem, Dcm, Os, MemIf), ARXML debugging, RTE generation troubleshooting and CAN COM-stack debugging.

misra

Audit C against MISRA C:2025 or write code that is compliant by construction.

code-review

Embedded C correctness review — ISR safety, integer overflow, races, stack — plus AUTOSAR naming-convention audit.

requirements

Elicit, refine and trace requirements in EARS notation with ASIL attributes.

iso26262

Run a HARA to assign ASIL, or derive Safety Goals and Functional Safety Requirements with FTTI.

embedded-testing

Generate MC/DC-covering unit tests and systematic boundary-value test points.

embedded-debugging

Triage a field problem report or debug a specific fault — HardFault, watchdog, Dem event, stack overflow.

change-and-impact

Analyze a change request before work begins, or trace ripple effects with a regression scope.

Install — Claude Code skills for embedded automotive, in one command

# Install all skills (global)
npx skills add ptsilivis/autonomousguy -g

# Browse what's inside before installing
npx skills add ptsilivis/autonomousguy --list

# Install one skill / target a specific agent
npx skills add ptsilivis/autonomousguy --skill autosar-bsw
npx skills add ptsilivis/autonomousguy -a claude-code
